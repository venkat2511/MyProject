Declaring a variable in javascript:
In javascript variables can be declared using three ways:
 -> var
 -> let
 -> const

var :
var is traditional way of declaring variables in javascript.
Variables declared with var are global or function scoped.
var variables can be redeclared and updated.

let:
It is enhanced version of var.
let variables are block scoped.
variables declared with let cannot be redeclared but updated.

const:
const is block scoped.
const variables cannot be redeclared and updated.
